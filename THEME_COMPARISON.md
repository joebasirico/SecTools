# Theme Comparison: Before & After

## Before (Clean Modern UI)
- White/light grey backgrounds
- Blue accent colors
- Sans-serif fonts (system default)
- Subtle shadows
- Professional corporate look
- Material Design inspired

## After (Hacker Terminal UI)
- Dark grey/black backgrounds
- Green terminal colors
- Monospace fonts (Courier New)
- Glowing shadows
- Cyberpunk/hacker aesthetic
- Terminal/console inspired

## Visual Elements Comparison

### Header
**Before:**
- Simple text "SecTools"
- Blue subtitle
- Minimal styling

**After:**
- ASCII art logo
- Terminal brackets [SECURITY TESTING FRAMEWORK]
- System status bar with metrics
- Pulsing cursor animation
- Glowing text effects

### Tool Cards
**Before:**
- White cards
- Blue "Use Tool" button
- Drop shadows
- Clean typography

**After:**
- Dark grey cards
- Green glowing borders
- Scan-line pulsing animation
- Terminal arrow indicator (▶)
- "EXECUTE TOOL" with cursor
- Hover glow effects

### Forms
**Before:**
- White background
- Grey borders
- Blue focus rings
- Standard input fields

**After:**
- Dark background
- Green borders
- Green glow on focus
- Terminal prompt (>) labels
- Monospace input text
- [EXECUTE ANALYSIS] button

### Results Display
**Before:**
- Light grey background
- Black text
- Blue/red accent colors
- List bullets

**After:**
- Dark card with green border
- Green terminal text
- Red/yellow severity colors
- Bullet points (•)
- Monospace code display

### Typography
**Before:**
- System fonts
- Standard weights
- Normal spacing

**After:**
- Courier New (monospace)
- Bold terminal headers
- Wider letter spacing
- Uppercase categories

### Navigation
**Before:** Standard links
**After:** Terminal-style navigation with arrows and brackets

### Feedback
**Before:** Flash messages in colored boxes

### Status Indicators
**Before:** Badges with background colors
**After:** Bracketed labels [CATEGORY NAME]

### Interactive Elements
**Before:** Smooth transitions, subtle hover states
**After:** Glow effects, pulsing animations, terminal cursor
